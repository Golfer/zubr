ZUBR
====

Run ZUBR Appication

`````
rake start
`````

Run same parsers

`````
curl http://localhost:4567/{name_what_you_are_needed}
`````

When you  needs to use params on url you need to add this parameters on example

`````
curl http://localhost:4567/route_url?url={params}
`````
